# Project Status

# Installation

```sh
gleam add sqlc_gen_gleam@1
```
```sh
$ gleam run -m sqlc_gen_gleam
```

Further documentation can be found at <https://hexdocs.pm/sqlc_gen_gleam>.

# Development

```sh
gleam run   # Run the project
gleam test  # Run the tests
```
